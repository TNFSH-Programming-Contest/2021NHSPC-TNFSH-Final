# pE. 遞迴呀遞迴
給定 $a$, $b$, $c$, $r_0$, $r_1$, $n$, $p$ 和遞迴關係式 $r_{i + 2} = a \cdot r_{i + 1} \cdot r_i + b \cdot r_{i + 1} + c \cdot r_i \ (i \ge 0)$，\
求 $r_n \pmod p$。

## 輸入
第一行有三個數字 $a, b, c$。\
第二行有二個數字 $r_0, r_1$。\
第三行有二個數字 $n, p$。

## 輸出
輸出 $r_n \pmod p$。

## 輸入範圍
- $-10^9 \leq a, b, c, r_0, r_1 \leq 10^9$
- $1 \le n \le 10^{18}$
- $1 \le p \le 10^4$

## 子任務
\subtasks

## 範例輸入 1
```
1 5 -1
0 -1
9 21
```

## 範例輸出 1
```
12
```

## 範例輸入 2
```
3 -3 5
6 6
10 72
```

## 範例輸出 2
```
30
```

## 範例輸入 3
```
-3 -5 2
9 9
4 35
```

## 範例輸出 3
```
20
```
