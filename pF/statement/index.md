行末使用兩個半形空格為換行語法。  

請問有多少個長度為 $n$ 的數列符合 $a_1 \le a_2 \ge a_3 \le a_4 \ge a_5$ ..... ，且 $0 \le a_i \le 9$ 呢？

\clearpage

## 輸入
一個整數 $n$

## 輸出
輸出一個整數 $ans$，代表答案 mod 1000000007 的結果。

## 輸入限制
 - $1 \leq n \leq 10^{18}$

## 子任務
\subtasks

\clearpage

## 範例輸入1
```
2
```

## 範例輸出1
```
55
```

## 範例輸入2
```
3
```

## 範例輸出2
```
385
```

## 範例輸入3
```
198964
```

## 範例輸出3
```
679096590
```
